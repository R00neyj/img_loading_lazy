# img_loading_lazy (v1.1)

HTML/PHP 내 `<img>` 태그를 자동으로 최적화해주는 팀 프로젝트 도구임.

## 주요 기능
- **자동화**: `input` 폴더의 모든 파일을 한 번에 처리해서 `output`으로 내보냄.
- **안정성**: PHP 코드(`<?php ... ?>`)와 주석을 완벽하게 보호하며 변환함.
- **정확성**: Cheerio 파서를 사용하여 HTML 구조가 깨지지 않음.
- **최적화**: `loading="lazy"`, 실제 이미지 사이즈 기반 `width`/`height`/`rem` 스타일 자동 삽입.

## 폴더 구조
- `input/`: 변환할 원본 `.html` 또는 `.php` 파일을 넣는 곳
- `output/`: 변환이 완료된 파일이 생성되는 곳
- `run.bat`: 스크립트 실행기 (더블 클릭!)

## 사용법
1. 프로젝트 폴더의 `run.bat` 파일을 더블 클릭함.
2. (최초 실행 시) 필요한 라이브러리가 자동으로 설치됨.
3. `input` 폴더에 파일이 없으면 폴더만 생성되고 종료됨.
4. `input`에 파일을 넣고 다시 `run.bat`을 실행하면 `output` 폴더에 결과물이 나옴.

## 주의 사항
- 이미지 경로(`src`)가 올바르지 않으면 이미지 사이즈를 읽어오지 못함 (사이즈 속성만 생략됨).
- Node.js가 컴퓨터에 설치되어 있어야 함.
